# MongoDB--小鸟程序员

### 一、MongoDB的基本概念

| mysql    | mongodb    | 解析          |
| -------- | ---------- | ------------- |
| database | database   | 数据库        |
| table    | collection | 数据库表/集合 |
| row      | document   | 行/文档       |
| column   | field      | 列/域         |

### 二、MongoDB常见数据类型

| 数据类型  | 解释                 |
| --------- | -------------------- |
| String    | 字符串 使用UTF-8编码 |
| Integer   | 整数类型             |
| Boolean   | 布尔类型             |
| Double    | 浮点类型             |
| Array     | 数组类型             |
| TimeStamp | 时间戳               |
| Object    | 内嵌文档             |
| Null      | 空值                 |
| date      | 日期类型             |
